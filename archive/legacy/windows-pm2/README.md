# Windows PM2 Legacy

Estos scripts se conservan solo como referencia histórica del flujo Windows basado en PM2.

## Estado

- Soporte: descontinuado para instalaciones nuevas.
- Alcance: consulta, recuperación puntual o comparación con releases antiguos.
- Reemplazo vigente: flujo portable basado en `launcher.cmd`, `stop.cmd`, `uninstall.cmd` y `scripts/release/`.

## Contenido archivado

- `install.cmd`
- `start_app.cmd`
- `update.cmd`

No se deben volver a conectar al flujo principal ni usar como mecanismo de despliegue por defecto.
